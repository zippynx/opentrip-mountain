# Zippy Tour Service - Open Trip Gunung Premium 

Landing page modern dan responsif untuk penyedia layanan *open trip* gunung profesional. Website ini dirancang untuk memberikan informasi paket perjalanan, fasilitas, dan keunggulan layanan dengan antarmuka yang menarik.

## 🚀 Fitur Utama
* **Desain Responsif:** Tampilan menyesuaikan dengan sempurna di layar HP, tablet, maupun desktop.
* **Animasi Interaktif:** Dilengkapi efek *scroll animation* (*fade up & fade in*) dan *floating elements* yang memanjakan mata.
* **UI/UX Modern:** Menggunakan elemen desain *Glassmorphism* pada *navbar* dan kartu layanan.
* **Mobile-Friendly Navigation:** Menu *hamburger* yang praktis untuk pengguna *smartphone*.

## 🛠️ Teknologi yang Digunakan
* **Struktur:** HTML5
* **Styling:** CSS3 & [Tailwind CSS](https://tailwindcss.com/) (via CDN)
* **Interaktivitas:** Vanilla JavaScript
* **Aset Visual:** [Font Awesome 6](https://fontawesome.com/) (Ikon) & [Google Fonts - Poppins](https://fonts.google.com/specimen/Poppins)
